Dataset

The dataset we have decided to analyse collects data relating to the measurements of air pollutants in the municipality of Milan. This data is freely available from https://web.comune.milano.it/cortesia/opendata.html. The following pollutants' concentrations are measured:

Sign Name
NO2 Nitrogen Dioxide
CO_8h Carbon monoxide
C6H6 Benzene
PM10 <10 micrometer particles
O3 Ozone
PM25 <2.5 micrometer particles
SO2 Sulfur dioxide

Preprocessing and Statistics

The data is organized in different datasets according to the year the measurements were made in. We are looking into measurements from 2015-2020. There are 8 different air pollutant levels recorded in the datasets: The datasets are in the following format:

stazione_id data inquinante valore
0 2 2015-01-01 SO2 2.5
1 2 2015-01-02 SO2 2.5
2 2 2015-01-03 SO2 2.5
3 2 2015-01-04 SO2 2.5
4 2 2015-01-05 SO2 5

The datasets consider the following features:

  • ID of the station where the measurement was made;
  • date of the measurement (note: measurements are made only during workdays);
  • name of pollutant;
  • value of the measurement in g/m^3. It must be noted that a pollutant’s concentration can be measured multiples times a day, sometimes with no regular pattern. Furthermore, the ‘value’ column contains a high number of missing values (as shown further down).

Each year’s ds contains a somewhat balanced number of observations, between 5700 and 6300, however there are a number of missing values. After dropping the missing values the following number of observation was left:

entries
NO2 9070
CO_8h 4985
C6H6 4300
PM10 4212
O3 3470
PM25 2640
SO2 1708
